0단계 작업을 시작해줘.

목표:
v0.5 기획서를 기준으로 Google Spreadsheet 데이터베이스 구조를 초기화한다.

구현 범위:
1. Constants.gs를 생성한다.
2. Database.gs를 생성한다.
3. 모든 시트명과 컬럼명을 상수로 정의한다.
4. setupDatabase() 함수를 구현한다.
5. seedMasterData() 함수를 구현한다.
6. resetDatabaseForDev() 함수를 구현하되, 개발용이며 기존 데이터를 삭제할 수 있음을 주석으로 명확히 표시한다.
7. 사진 업로드/카메라 기능은 제외한다.
8. 마법공격력 관련 컬럼과 변수는 만들지 않는다.

Effects 기본 데이터:

- debuff_freeze: 빙결
- debuff_burn: 발화
- debuff_stun: 기절
- debuff_weak: 약화, attack percent -25
- debuff_corrosion: 부식, defense percent -33
- debuff_dazed: 멍해짐, questionTime flat -3
- debuff_foolish: 멍청해짐, questionDifficulty flat +1
- buff_power: 힘, attack flat n, stackable true
- buff_hard: 단단함, defense flat n, stackable true
- buff_focus: 집중, criticalRate flat n, stackable true
- buff_smart: 똑똑해짐, questionDifficulty flat -1, minimum difficulty 1
- buff_wisdom: 지혜, questionTime flat +n
